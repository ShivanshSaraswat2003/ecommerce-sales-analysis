# 📊 E-commerce Sales & Logistics Analysis


## 🎯 Objective
Analyze sales performance, customer behavior, and delivery efficiency using Python and Power BI.

---

## 🛠️ Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Excel
- Power BI
  
## 🚀 Project Highlights
- Analyzed 9,800+ e-commerce transactions
- Performed data cleaning, feature engineering, and EDA using Python
- Identified key revenue drivers and seasonal trends
- Detected delivery inefficiencies using shipping analysis
---

## 📈 Key Insights
- West region generated highest revenue
- Consumer segment contributed ~50% of total sales
- Technology category dominated performance
- Seasonal spikes observed during Sep–Dec
- Delivery delays identified in 5–6 day range

---

## 🚚 Logistics Analysis
- Average shipping time: ~4 days
- Delay patterns observed beyond 5 days
- Same-day deliveries detected (potential anomalies)
  
## 📊 Key Metrics
- Total Sales: $2,252,607
- Total Orders: 4,916
- Average Shipping Time: ~4 days
- Top Region: West
- Top Segment: Consumer
  
- ## 📊 Key Business Recommendations
- Optimize logistics in regions with 5–6 day delays
- Focus marketing on Consumer segment (highest contribution)
- Increase inventory for Technology category
- Plan campaigns around Sep–Dec seasonal spike

  # 🚀 Final Conclusion
- The analysis shows that revenue is primarily driven by the West region and Consumer segment.
- Technology category dominates sales performance.
- Seasonal spikes during Sep–Dec indicate strong demand patterns.
- Shipping delays beyond 5 days highlight operational inefficiencies.
